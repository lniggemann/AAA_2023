# AAA_2023

## How2Run

requirements:
- python version 3.10
- pipenv


from terminal:
- run "pipenv install" to install virtual env
- run "pipenv shell" to activate virtual env

reset environment:
- delete vitual env folder (location win: "C:\Users\XYZ\\.virtualenvs\AAA_2023-xxxxxxx")
- rerun pipenv install and pipenv shell
